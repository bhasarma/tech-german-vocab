# Tech German Vocabulary for AI and Data Professions

This repository provides most used German vocabulary for professionals in AI, data science, software development, and related tech fields, helping non-native speakers navigate in the German tech industry.

## VERBS

| **S.No.** | **Verb**           | **Meaning (English)**        |  **Example Sentences** |
| --------- | ------------------ | ---------------------------- | ---------------------- |
| 1   | abbrechen        | to cancel, to terminate              | **1**. Wenn das Modell während des Trainings keine Verbesserungen zeigt, kann es sinnvoll sein, das Training **abzubrechen** und die Hyperparameter anzupassen. <br> (If the model shows no improvement during training, it may be useful to stop the training and adjust the hyperparameters.)<br> **2**. Wenn die Datenqualität nicht den Anforderungen entspricht, sollten Sie den ETL-Prozess **abbrechen** und die Datenquellen überprüfen. <br> (If the data quality does not meet the requirements, you should stop the ETL process and review the data sources.)<br> **3**. Das Experiment wurde aufgrund eines Fehlers in den Berechnungen **abgebrochen**.<br> (The experiment was terminated due to an error in the computations.) <br> **4**. Falls die API-Anfrage zu lange dauert, können Sie den Prozess **abbrechen**, um Zeit und Ressourcen zu sparen. <br> (If the API request takes too long, you can cancel the process to save time and resources.) <br> **5**. Die Datenvorverarbeitung wurde **abgebrochen**, da der Speicherplatz auf dem Server erschöpft war.<br> (The data preprocessing was stopped because the server ran out of storage space.)|
| 2   | abfragen         | to query                             | **1**. Der Algorithmus wurde entwickelt, um eine große Datenbank effizient **abzufragen** und relevante Informationen in Echtzeit zu liefern.<br> (The algorithm was designed to query a large database efficiently and deliver relevant information in real time.)<br> **2**. Um das Modell zu evaluieren, müssen wir die Testdaten aus der Datenbank **abfragen** und die Vorhersagen vergleichen.<br>(To evaluate the model, we need to query the test data from the database and compare the predictions.)<br>**3**. In einem Machine-Learning-Workflow werden die Trainingsdaten häufig von einem Server **abgefragt**, bevor sie verarbeitet und das Modell trainiert wird.<br>(In a machine learning workflow, the training data is often queried from a server before it is processed and used to train the model.)<br> **4**. Die API kann verwendet werden, um Echtzeit-Daten von verschiedenen Sensoren **abzufragen** und in das Machine-Learning-Modell einzuspeisen.<br>(The API can be used to query real-time data from various sensors and feed it into the machine learning model.)<br> **5**.  Bevor die Datenanalysen beginnen, müssen wir sicherstellen, dass wir die korrekten Daten aus dem Data Warehouse **abfragen**.<br>(Before starting the data analysis, we must ensure that we are querying the correct data from the data warehouse.)|
| 3   | abrufen          | to retrieve, to fetch                | **1**. Das Modell kann aktuelle Wetterdaten von einer API **abrufen** und diese für Vorhersagen verwenden.<br> (The model can retrieve current weather data from an API and use it for predictions.) <br>**2**. Um die neuesten Kundendaten zu analysieren, müssen wir die Daten aus dem Data Warehouse abrufen.<br> (To analyze the latest customer data, we need to retrieve the data from the data warehouse.) <br>**3**. Der Machine-Learning-Algorithmus ruft automatisch neue Daten vom Server ab, um kontinuierlich zu lernen und seine Genauigkeit zu verbessern.<br> (The machine learning algorithm automatically retrieves new data from the server to continuously learn and improve its accuracy.) <br>**4**.  In einem Echtzeitsystem müssen Modelle schnell Daten abrufen, um sofortige Entscheidungen treffen zu können.<br> ( In a real-time system, models must quickly retrieve data to make instant decisions.) <br> **5**. Bevor das neuronale Netzwerk trainiert werden kann, müssen wir die Trainingsdaten aus der Cloud abrufen.<br> (Before the neural network can be trained, we need to retrieve the training data from the cloud.)|
| 4   | abschätzen       | to estimate                          | **1**. Um den Erfolg eines Machine-Learning-Modells zu bewerten, müssen wir die Genauigkeit und den Fehler des Modells **abschätzen**.<br>(To evaluate the success of a machine learning model, we need to estimate its accuracy and error.)<br> **2**. Bei der Datenanalyse ist es wichtig, den Einfluss fehlender Daten korrekt **abzuschätzen**, um Verzerrungen zu vermeiden.<br>(In data analysis, it is important to correctly estimate the impact of missing data to avoid bias.)<br> **3**. Der Algorithmus kann die zukünftige Nachfrage abschätzen, basierend auf historischen Verkaufsdaten und aktuellen Trends.<br> (The algorithm can estimate future demand based on historical sales data and current trends.) **4**. Um das Risiko eines Projekts zu bewerten, muss der Analyst die Unsicherheiten und deren Auswirkungen auf das Ergebnis abschätzen.<br> (To assess the risk of a project, the analyst must estimate the uncertainties and their impact on the outcome.) <br> **5**. In der künstlichen Intelligenz wird häufig der Ressourcenverbrauch eines Modells abgeschätzt, bevor es auf einer größeren Datenmenge trainiert wird. <br> (In artificial intelligence, the resource consumption of a model is often estimated before it is trained on a larger dataset.)|
